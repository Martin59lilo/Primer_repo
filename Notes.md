# Commands
## Terminal
```
ls
ls -a
cd
cd ..
mkdir
rm
mv archivo destino
```
Flujo de trabajo:
- Computador
- Stage
- Commit
- Server
## Git
To stage:
```
git status
git add nombre del archivo nombre del archivo
git add .
git rm nombre del archivo
git mv archivo destino
git restore --staged nombre del archivo
git restore
```
To commit:
```
git commit -m "Mensaje"
git commit
```
